# Socio-Economic Impact Analysis: Rural vs Urban BMI Trends

## Project Overview
This project demonstrates advanced statistical analysis using **STATA**. It explores the relationship between geographic residency (Rural vs. Urban) and Body Mass Index (BMI) using the NHANES II dataset.

## Research Questions
1. Does residency significantly impact health outcomes (BMI)?
2. Does this impact hold when controlling for demographic factors like Age and Sex?

## Technical Skills Demonstrated
- **Data Management:** Loading web-based datasets and variable recoding.
- **Statistical Modeling:** Implementation of OLS (Ordinary Least Squares) Regression.
- **Econometric Controls:** Using multivariate models to isolate specific effects.
- **Professional Reporting:** Automating publication-ready tables using `outreg2`.

## Project Files
- `SocioEconomic_Analysis.do`: The full STATA script.
- `Final_SocioEconomic_Analysis.doc`: The generated research table.
- `stata_results_table.png`: A visual preview of the statistical output.

## Key Findings
The analysis found a statistically significant difference in BMI between rural and urban populations, which persisted even after adjusting for age and gender.
